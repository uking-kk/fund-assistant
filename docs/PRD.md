# 基金智能助手 - 产品需求文档 (PRD)

## 一、产品概述

### 1.1 产品定位

**基金智能助手**是一个基于大语言模型的基金投资问答机器人，帮助普通投资者解决"选基金难"的问题。

**核心价值**：
- 降低基金投资门槛
- 提供专业的投资建议
- 解读复杂的基金报告

### 1.2 目标用户

| 用户画像 | 痛点 | 需求 |
|---------|------|------|
| 投资新手 | 不懂基金基础知识 | 知识问答、概念解释 |
| 上班族 | 没时间研究 | 快速查询、智能推荐 |
| 基金持有者 | 看不懂基金报告 | 报告解读、持仓分析 |

### 1.3 核心功能

```
┌─────────────────────────────────────────────────────┐
│                    基金智能助手                      │
├─────────────────────────────────────────────────────┤
│                                                     │
│  💬 智能问答（核心）                                 │
│     └── 基金知识、投资策略、术语解释                 │
│                                                     │
│  📊 基金查询（辅助）                                 │
│     └── 净值查询、收益分析、基金对比                 │
│                                                     │
│  📄 报告解读（辅助）                                 │
│     └── 季报/年报上传、关键信息提取                  │
│                                                     │
│  🎯 智能推荐（辅助）                                 │
│     └── 根据偏好推荐基金                            │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## 二、功能详细设计

### 2.1 智能问答（核心功能）

#### 功能描述
用户通过自然语言提问，AI基于知识库给出专业回答。

#### 用户场景

| 场景 | 用户问题 | AI回答 |
|------|---------|--------|
| 概念学习 | "什么是ETF基金？" | 解释ETF的定义、特点、优缺点 |
| 策略咨询 | "定投好还是一次性买入？" | 对比两种策略的适用场景 |
| 操作指导 | "如何看基金评级？" | 介绍主流评级机构和查看方法 |
| 风险咨询 | "债券基金有风险吗？" | 解释债券基金的风险类型 |

#### 技术实现
- **RAG（检索增强生成）**：从知识库检索相关内容，结合LLM生成回答
- **知识库内容**：基金基础知识、投资策略、常见问题FAQ

#### 交互设计
```
用户: 什么是ETF基金？

AI: ETF（Exchange Traded Fund）即交易型开放式指数基金，
    是一种跟踪指数、可以在交易所买卖的基金。

    📌 ETF的特点：
    • 被动跟踪指数，费用低
    • 可以像股票一样实时买卖
    • 分散投资，降低风险

    💡 适合人群：想投资指数但不想选股的投资者

    还有其他问题吗？
```

---

### 2.2 基金查询（辅助功能）

#### 功能描述
查询具体基金的数据信息，包括净值、收益、持仓等。

#### 用户场景

| 场景 | 用户问题 | 系统行为 |
|------|---------|---------|
| 单只查询 | "查一下易方达蓝筹精选" | 返回基金基本信息卡片 |
| 收益查询 | "这只基金最近一年收益怎么样" | 返回收益曲线图 |
| 对比查询 | "对比这几只基金" | 返回对比表格 |
| 持仓查询 | "这只基金重仓了什么" | 返回持仓列表 |

#### 数据来源
天天基金网公开API（免费，无需认证）

#### 交互设计
```
用户: 帮我查一下易方达蓝筹精选

AI: 好的，这是这只基金的信息：

    ┌──────────────────────────────────────┐
    │ 📊 易方达蓝筹精选混合 (005827)        │
    ├──────────────────────────────────────┤
    │ 近1年收益: +12.35%                   │
    │ 近3年收益: +45.67%                   │
    │ 基金经理: 张坤                       │
    │ 基金规模: 678.92亿                   │
    │ 风险等级: 中高风险                   │
    │ 基金类型: 混合型                     │
    └──────────────────────────────────────┘

    这只基金由明星基金经理张坤管理，
    长期业绩优秀，适合长期持有。
```

---

### 2.3 报告解读（辅助功能）

#### 功能描述
用户上传基金季报/年报PDF，AI提取关键信息并回答问题。

#### 用户场景

| 场景 | 用户行为 | AI输出 |
|------|---------|--------|
| 上传报告 | 上传PDF文件 | 自动提取关键信息摘要 |
| 持仓分析 | "这只基金重仓了哪些股票？" | 返回前十大重仓股 |
| 业绩归因 | "为什么业绩下滑？" | 分析业绩归因 |
| 经理观点 | "基金经理怎么看后市？" | 提取经理观点 |

#### 技术实现
- **文档解析**：PyPDF2 / pdfplumber 解析PDF
- **向量化存储**：将报告内容向量化存入Chroma
- **RAG检索**：基于用户问题检索相关段落

---

### 2.4 智能推荐（辅助功能）

#### 功能描述
根据用户的风险偏好和收益目标，推荐合适的基金。

#### 用户场景

```
用户: 我想找一只年化5%左右的稳健基金

AI: 根据您的需求，我为您推荐以下基金：

    ┌──────────────────────────────────────┐
    │ 🏆 推荐基金：易方达增强回报债券A      │
    ├──────────────────────────────────────┤
    │ 近1年收益: 5.23%                     │
    │ 近3年收益: 16.78%                    │
    │ 风险等级: 中低风险                   │
    │ 推荐理由: 纯债基金，波动小，收益稳定  │
    └──────────────────────────────────────┘

    这只基金主要投资债券，风险较低，
    适合追求稳健收益的投资者。
```

## 三、非功能需求

### 3.1 性能要求

| 指标 | 要求 |
|------|------|
| 响应时间 | 首次响应 < 3秒，流式输出 |
| 并发支持 | 支持10个用户同时使用 |
| 可用性 | 99% 在线时间 |

### 3.2 安全要求

| 要求 | 实现方式 |
|------|---------|
| API密钥保护 | 环境变量存储，不提交代码库 |
| 用户数据 | 不存储敏感信息，对话不持久化 |
| 输入验证 | 过滤恶意输入，限制文件大小 |

### 3.3 部署要求

| 环境 | 平台 | 说明 |
|------|------|------|
| 前端 | Vercel | 免费额度，自动部署 |
| 后端 | Railway / Render | 免费额度，支持Python |
| 数据库 | Chroma本地 / Pinecone云端 | 向量数据库 |
